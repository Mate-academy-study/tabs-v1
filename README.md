# Tabs Monorepo

Monorepo with "tabs" component in two versions: vanilla JavaScript and React.

## Installation

```bash
npm install
```

## Running

```bash
npm run dev -w vanilla    # Vanilla JS
npm run dev -w react      # React
```

## Structure

```
packages/
├── vanilla/    # Pure JavaScript + Vite
└── react/      # React 19 + Vite
```
