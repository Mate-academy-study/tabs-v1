# Tabs Monorepo

Monorepo z komponentem "tabs" w dwóch wersjach: vanilla JavaScript i React.

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev -w vanilla    # Vanilla JS
npm run dev -w react      # React
```

## Struktura

```
packages/
├── vanilla/    # Czysty JavaScript + Vite
└── react/      # React 19 + Vite
```
